# Emotion Detection Frontend

A modern React application for real-time emotion detection using AI, built with:

- ⚡ Vite
- 🏗️ TypeScript
- 🎨 Tailwind CSS
- 📊 Recharts

## Features

- Real-time camera emotion analysis
- Image capture and processing
- Emotion history dashboard
- Responsive design

## Quick Start

### Installation
```bash
npm install
```
### Development
```bash
npm run dev
```
